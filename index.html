This option set directly changes values in the database:

Item Identify List works in conjunction with Item Identify Data to determine how CHEC handles data collected from a scanner.

Item Identify List defines the sorting algorithm that identifies into which Item Identify Data bucket the scan data belongs.
Item Identify Data works in conjunction with Item Identify List to determine how CHEC handles data collected from a scanner.

Item Identify Data defines the Item Processing Type, Special Handling Instructions, Assistant Required, and Exclusive Mode (full-lane only).
Store Configurations allow configuration of the replication intervals, replication "quiet time", the local currency and measurement system for the store.
BOSS Configurations allow configuration of the History Analysis Profile and Data Retention Policies.
Consolidated Item Security allows configuration of PLU ranges to be included in the consolidated item security profiles at the Enterprise BOSS level.
